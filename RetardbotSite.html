Running the Bot Yourself

Downloading the Code
Head on over to https://github.com/ah-yis/RetardBot
Click on the big green "Code" button, and select "Download ZIP"
Save it anywhere you want, and then extract it to its own folder.


Downloading Dependencies
Just head over to https://nodejs.org/en/download and grab the right installer.
There's some other things you'll need, like FFMPEG. I'm not sure how it's installed anywhere else, but Windows steps are provided.
Also, this isn't a dependency, but it is recommended to use something like Visual Studio Code to run the bot.
Also, you need to know beforehand how to make discord bot invite links, and you must have used this official guide.


Setting Up FFMPEG
After installing the binaries on Windows (i cant help you with anything else), you need to put them in a permanent folder somewhere.
Now open up Start and search up "View advanced system settings". Head on over to "Environment Variables" and find "Path" under "System variables".
Select "Edit...", click "New", and paste in the path of the folder with your FFMPEG binaries. This is important if you want to use voice channel stuff.


The Final Steps
Now, if you're using Visual Studio Code, all you have to do is press 'CTRL + K' and then immediately 'CTRL + L'.
Now, select the folder where you extracted all the code.
You are not done yet. Open up "main.js" and then, on (by default) line 14, enter your token into "client.login('');"
